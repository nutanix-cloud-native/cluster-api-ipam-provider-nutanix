# CAPI IPAM Provider for Nutanix documentation site

The docs site uses [hugo](https://gohugo.io/) to build a static site deployed to the `gh-pages` branch of this repo.

## Creating a new page

`hugo new content/<kebab-case-page-name>.md`

## Previewing the site

`hugo serve -F -D`
